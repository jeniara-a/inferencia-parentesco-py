Inferência de Grau de Parentesco com IA

Uma solução inteligente para analisar relações familiares

🔍 Sobre o projeto

Este projeto aplica técnicas de aprendizado de máquina para inferir automaticamente o grau de parentesco (como filho, cônjuge ou nenhum) entre indivíduos com base em dados e similaridade de nomes.

Ver relatório

⚙️ Como funciona

  • Pré-processamento e normalização dos nomes
  • Extração de features com base em similaridade
  • Treinamento de um modelo RandomForestClassifier
  • Inferência de grau de parentesco com combinação de múltiplas predições

📁 Estrutura

  • /src - Código fonte
  • /data - Dataset usado para treino
  • /docs - Arquivos para o GitHub Pages
